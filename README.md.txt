1. Run SQL script (studentcourseDB.sql) in your HeidiSQL to create database and tables and insert all the related data 
2. Open project source code in your IDE and start running SMSRunner in jpa.mainrunner package
3. It will appear read line in your ide condole which is normal (Hibernate is doing its job) and just follow the prompts to enter selection 
4. It will automatically quit the application when u done register to any class 
5. There is one junit test for one method in src/test/java -> jpa.test -> StudentDAOTest
6. You can re-run and test the application again.
